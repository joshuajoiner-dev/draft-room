# Draft Room v1.0 Release Notes

Draft Room v1.0 is a live team creation room for classrooms, sports, camps, clubs, and events.

## Features Included

- Create a room.
- Share a room code, join link, and QR code.
- Copy room code and join link.
- Players join by name.
- Admin imports player names by line, comma, or tab-separated paste.
- Duplicate imported names are skipped.
- Live room player list updates.
- Quick Random team creation.
- Balanced Random team creation with even team sizes.
- Captain Draft setup with captains assigned to their own teams.
- Captain Draft picking by round.
- Admin override controls for unassign, reassign, and supported undo actions.
- Finalized-room lock behavior at the override layer.
- Mobile-first dark UI.

## Supported Workflows

- Classroom group creation.
- Camp or club team creation.
- Sports practice team creation.
- Event participant sorting.
- Captain-led drafting from a shared room pool.
- Admin correction of team assignments before final use.

## Known Limitations

- No user accounts or authentication.
- Anyone with the admin URL can use admin controls.
- No short custom join URL.
- No export workflow.
- No persistent room history page.
- Balanced Random balances team size only; it does not use skill ratings or categories.
- Captain Draft does not enforce captain identity beyond the current room UI.

## Future Ideas

- Authenticated admin access.
- Short join URLs.
- Export teams to CSV or PDF.
- Optional room history for admins.
- Additional display mode for projection in large rooms.
- More robust audit trail for admin overrides.
